01 Path Planner for Differential Drive Robots
Ongoing · ROS2 · C++ · Nav2 Plugin

A from-scratch path planning algorithm for differential drive robots, implemented as a drop-in Nav2 global planner plugin in C++. The planner generates smooth, curvature-continuous trajectories using spline interpolation with non-holonomic curvature bounds, adaptive lookahead distance, and dynamic velocity profiling tuned to the robot's physical constraints.

The implementation integrates directly with Nav2's lifecycle management and the Costmap2D infrastructure, allowing seamless drop-in replacement in any standard ROS2 navigation setup without changes to the broader stack.

02 Path Planner for 3-Wheel Omnidirectional (Swerve) Drive
Ongoing · ROS2 · C++ · Holonomic Kinematics

Extension of the path planner to the kinematically richer three-wheel omnidirectional swerve configuration — a holonomic platform capable of simultaneous independent translation and rotation. The work involves custom kinematic modelling of the 3WD geometry, holonomic velocity decomposition into per-wheel speed and heading commands, and heading-independent motion planning.

The planner supports fully decoupled heading control — the robot can translate in any direction while maintaining an arbitrary field-relative orientation — which is critical for competition tasks requiring precise positioning while facing a fixed target.

03 Robocon Autonomous Navigation Stack
DD Robocon 2024–2025 · ROS2 · Full Stack

Complete ROS2 autonomous navigation stack built for and deployed on competition robots at DD Robocon. SLAM Toolbox runs online to build a live occupancy grid of the competition field. Once a map is established, AMCL provides probabilistic Monte Carlo localisation within it. An Extended Kalman Filter from the robot_localization package fuses wheel odometry and IMU data into a robust pose estimate.

Nav2's BehaviorTree-based mission executor handles high-level task sequencing. The full stack was tuned, tested, and deployed on hardware under competition conditions — contributing directly to the National Championship win in 2024 and the Runners Up finish in 2025.

04 Competition Robot Mechanical Design (SolidWorks)
SolidWorks · CAD Assembly · Drivetrain Engineering

End-to-end mechanical design of Robocon competition robots in SolidWorks. Responsible for full parametric CAD assembly — chassis layout and structural frame, custom drivetrain design for both differential and swerve configurations, bearing and shaft sizing, fastener selection, and subsystem interfaces.

Designs are developed against competition constraints — footprint limits, weight budgets, and field interaction requirements — with manufacturing tolerances embedded throughout. Final assemblies are fabricated directly and fielded in competition.
