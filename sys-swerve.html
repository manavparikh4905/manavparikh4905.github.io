Path Planner for 3-Wheel Omnidirectional (Swerve) Drive

Demonstration of independent translation and rotation maneuvering.

Problem

Differential drives are constrained by heading-dependent translation, making precise positioning around obstacles slow as it requires stop-and-turn mechanics.

Approach

Developed a custom path planner leveraging the holonomic kinematics of a 3-Wheel Omnidirectional (Swerve) drive, allowing independent translation and rotation.

System Architecture

Laptop (ROS2) ↓ Nav2 Planner Server ↓ Twist (Vx, Vy, Vθ) ↓ Kinematics Node (3WD Decomposition) ↓ Motor Speeds & Steering Angles ↓ STM32 Microcontroller ↓ Swerve Modules

Implementation

C++ node translating global Twist commands into independent wheel speeds and orientations. The planner fully decouples movement from the robot heading, computing inverse kinematics for the 3WD geometry.

Result

Robot translates freely in any direction while concurrently reorienting to face targets, drastically improving competition task efficiency.
