Robocon Autonomous Navigation Stack

Full navigation stack executing a complete field run autonomously.

Problem

The competition field requires robust, online mapping and extremely reliable localization despite dynamic obstacles, collisions, and wheel slip.

Approach

Integrated a full ROS2 Navigation stack using SLAM Toolbox for cartography, AMCL for probabilistic localization, and an Extended Kalman Filter (EKF) for sensor fusion.

System Architecture

Lidar Wheel Encoders BNO055 IMU ↓ ↓ ↓ SLAM Toolbox robot_localization EKF (Sensor Fusion) ↓ ↓ ↓ Occupancy Grid Odom Base Link \ | / \ | / \ | / AMCL (Map to Odom Transform) ↓ Nav2 Behavior Tree ↓ Costmap & Planners

Implementation

Tuned map resolution, EKF covariance matrices, and particle filter parameters for the high-acceleration environment of Robocon. Engineered custom behavior trees using BehaviorTree.CPP to manage task sequences and recovery behaviors.

Result

Consistent sub-centimeter localization enabling repeatable high-speed autonomous runs across various configurations.
