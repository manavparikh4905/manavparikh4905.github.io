Hybrid A* Path Planner for Differential Drive Robots

Demonstration of the planner generating feasible trajectories.

Problem

Standard A* produces paths that violate differential drive kinematics. It relies on a discrete grid, leading to zig-zag paths that cannot be followed smoothly without halting to turn.

Approach

Implemented Hybrid A* with continuous state expansion (x, y, θ) and Dubins primitives to ensure kinematic feasibility and curvature bounds.

System Architecture

Laptop (ROS2) ↓ AMCL Localization ↓ Hybrid A* Nav2 Plugin ↓ Velocity Commands (Twist) ↓ Jetson Nano ↓ STM32 Controller ↓ Motor Drivers ↓ Robot Chassis

Implementation

Engineered natively as a custom ROS2 Nav2 global planner plugin in C++. It integrates directly with AMCL for probabilistic localization and standard Nav2 Costmap2D for collision checking.

Result

Robot successfully navigates highly cluttered environments using smooth, dynamically feasible trajectories without stop-and-turn mechanics.
